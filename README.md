# styleapp

A web application for managing and exploring personal style.

## Tech Stack

- **React 19** – UI library
- **Vite** – build tool & dev server
- **ESLint** – linter

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Available Scripts

| Script | Description |
|--------|-------------|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |

## Next Steps

Here are the recommended next steps to grow this project:

1. **Define the core features** – Decide what the app will do (e.g. outfit planner, wardrobe tracker, style inspiration feed).
2. **Set up routing** – Add [React Router](https://reactrouter.com/) (`npm install react-router-dom`) to support multiple pages.
3. **Add state management** – For complex state, consider [Zustand](https://zustand-demo.pmnd.rs/) or the built-in React Context API.
4. **Connect a backend / database** – Use [Supabase](https://supabase.com/) or [Firebase](https://firebase.google.com/) for authentication and data storage.
5. **Style the UI** – Integrate [Tailwind CSS](https://tailwindcss.com/) or a component library such as [shadcn/ui](https://ui.shadcn.com/).
6. **Add tests** – Set up [Vitest](https://vitest.dev/) for unit/component tests.
7. **Deploy** – Deploy to [Vercel](https://vercel.com/) or [Netlify](https://www.netlify.com/) with a single command.
